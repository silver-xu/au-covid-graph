COVID19 Statistics API API Reference

GraphQL API to retrieve the latest COVID19 infection statistics by country and region. The data updated daily from John Hopkins University at 1pm Australia Eastern time. The API was developed to primarily focus Australia COVID19 data but more support will be added to the rest of the world.

API Endpoints
Prod:
https://api.covidstats.com.au/graphql

Paths

Queries

Global

The top level object of the Covid Stats API. It returns the current global data of COVID 19 infections combined with history statistics

Example

Request Content-Types: application/json
Query
query global{
  global{
    totalConfirmedCases
    newlyConfirmedCases
    totalDeaths
    newDeaths
    totalRecoveredCases
    newlyRecoveredCases
    lastUpdatedDate
    history
  }
}

String: string

The String scalar type represents textual data, represented as UTF-8 character sequences. The String type is most often used by GraphQL to represent free-form human-readable text.
